# ODD

[\[\<\<\<\]](ug_25.148.md) [\[\>\>\>\]](ug_25.150.md)

Return `true` if the argument is an odd number. `ODD(undef)` is `undef`
or raises an error if the option `RaiseMatherror` is set in bit
`sbMathErrUndef`.

See also [EVEN](ug_25.54.md)

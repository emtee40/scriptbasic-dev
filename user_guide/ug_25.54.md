# EVEN

[\[\<\<\<\]](ug_25.53.md) [\[\>\>\>\]](ug_25.55.md)

Return `true` if the argument is an even number. `EVEN(undef)` is
`undef` or raises an error if the option `RaiseMatherror` is set in bit
`sbMathErrUndef`.

See also [ODD](ug_25.149.md).

# Getting the host name

[\[\<\<\<\]](ug_13.1.md) [\[\>\>\>\]](ug_14.md)

ScriptBasic can easily get the name of the host the code is running
calling the function `hostname`. This function returns the internet name
of the host. For example the code fragment

    print hostname,"\n"

prints `vphome` on my machine.
